## run app
1. Update composer
  - composer update

2. Install npm
  - npm install

3. Require passport
  - composer require laravel/passport

4. Run migration 
  - php artisan migrate
  - php artisan db:seed

5. Install passport
  - php artisan passport:install

6. Run php serve
  - php artisan serve

7. Run NPM
  - npm run dev
  - npm run prod
